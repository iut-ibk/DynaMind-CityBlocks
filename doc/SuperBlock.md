#SUPERBLOCK

creates a rectangular face.

##Parameter
| Name |  Type | Description| 
|------|-------|------------|
| NameOfSuperblock | string| name of the created face|
| Height | double | m |
| Width | double | m |
| offsetx | double | m |
| offsety | double | m |
| appendToExisting| bool | append face to existing view |
